Argumnts are values that get passed through the parameters into the function to be used.
For example, in the following function:
function vuTest(num1, num2) { console.log (vuTest)};

"function" is the keyword
"vuTest" is the function name
"num1" and "num2" are the parameters
When the function is called, the arguments of "1" and "2" are included like so:
vuTest(1,2);
